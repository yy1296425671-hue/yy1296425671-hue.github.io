## 设计风格
- 温柔质感风 — 暖奶油底色 (#faf7f2) + 暖白卡片 (#fffefb)
- 字体: Cormorant Garamond + Noto Serif SC（衬线体展示）+ DM Sans（标签）
- 主色调: 鼠尾草绿 (#7a9e6b) 作点缀
- 全页面纸张纹理覆盖

## 页面结构
- 导航: 作品 / 关于 / 联系（平滑滚动）
- Hero: 姓名 + 标题 + 阿里实习徽章 + 求职状态 + 简历下载
- Intro: 数据亮点条（3项目 / 独立全链路交付 / 阿里实习）
- 作品: ECLAT LEAF、REMORIE、PICKABOO 三个独立项目卡片（含亮点列表 + 工具标签 + 图片画廊 + 灯箱）
- 关于: 个人介绍 + 技能 + 时间线（阿里高亮）
- 联系: 邮箱 + 电话 + 站酷链接
